This repository contains the full analytical workflow supporting the manuscript:

“From forest to fields: The role of soil microbiome spillover in agroecosystem sustainability”
Submitted to Science Advances

Overview

This project investigates how surrounding forest cover influences the composition, diversity, assembly processes, functional potential, and ecosystem functioning of soil microbiomes in orchard systems.

The workflow integrates:
16S rRNA (bacteria) and ITS (fungi) sequencing data
Soil physicochemical properties
Landscape metrics (forest cover at multiple spatial scales)
Network analysis
Niche breadth and assembly processes
Soil respiration experiments

Main hypotheses:
1- Orchards with greater surrounding forest cover share more microbial taxa with adjacent forests (spillover).
2- Microbial community composition in orchards converges toward forest communities with increasing forest cover.
3- Forest cover influences:
      microbial assembly processes
      niche breadth distributions
      co-occurrence network structure
4- These microbiome differences affect soil heterotrophic respiration.
