# 🌱 From forest to fields...

This repository contains the full analytical workflow supporting the manuscript:

> **“From forest to fields: The role of soil microbiome spillover in agroecosystem sustainability”**  
> *Submitted to Science Advances*

---

## 📌 Overview

This project investigates how **surrounding forest cover** influences:

- microbial **composition**
- microbial **diversity**
- **assembly processes**
- **functional potential**
- **ecosystem functioning**

of soil microbiomes in orchard systems.

---

## 🔬 Integrated data and approaches

The workflow integrates:

- 🧬 **16S rRNA sequencing** (bacteria)  
- 🍄 **ITS sequencing** (fungi)  
- 🌱 **Soil physicochemical properties**  
- 🌍 **Landscape metrics** (forest cover at multiple spatial scales)  
- 🕸️ **Network analysis**  
- 🧠 **Niche breadth and community assembly**  
- 🌡️ **Soil respiration experiments**  

---

## 🧠 Main hypotheses

1. **Microbial spillover**  
   Orchards with greater surrounding forest cover share more microbial taxa with adjacent forests.

2. **Community convergence**  
   Microbial community composition in orchards becomes more similar to forest communities as forest cover increases.

3. **Ecological structuring**  
   Forest cover influences:
   - microbial assembly processes  
   - niche breadth distributions  
   - co-occurrence network structure  

4. **Ecosystem consequences**  
   These microbiome differences affect **soil heterotrophic respiration**.
